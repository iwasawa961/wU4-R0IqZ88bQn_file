FastDiffCopy Web
================

これは Smart App Control に引っかからない配布を目的にしたブラウザ版です。
GitHub Pages などの HTTPS 上に置くと、Edge / Chrome でそのまま使えます。

配置するファイル:
  outputs/web/index.html

GitHub Pages での公開例:
  1. GitHub に新しいリポジトリを作成します。
  2. index.html をリポジトリ直下へアップロードします。
  3. Settings -> Pages で main ブランチを公開元にします。
  4. 表示された https://...github.io/... のURLを Edge / Chrome で開きます。

ブラウザ版の仕様:
  ・コピー元は7つまで指定できます。
  ・コピー先はフォルダを指定します。
  ・コピー元にはフォルダまたはZIPファイルを指定できます。
  ・同名かつ同一サイズは確認なしでスキップします。
  ・同名でサイズが異なる場合だけ確認画面を表示します。
  ・確認画面では「コピー元を残す」「コピー先を残す」のチェックで処理を決めます。
  ・更新日時は比較に使いません。
  ・ブラウザ版はコピー専用です。

注意:
  ・ブラウザの安全制限により、コピー先ZIPを直接更新することはできません。
  ・ZIPはコピー元として読み込み、展開内容をコピー先フォルダへ整理します。
  ・ブラウザの削除権限は環境差が出やすいため、移動機能は外しています。
  ・空フォルダもコピー対象です。
  ・File System Access API を使うため、Edge / Chrome を推奨します。
